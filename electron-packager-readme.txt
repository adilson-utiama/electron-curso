1- npm install electron-packager -g
2- NO TERMINAL

electron-packager <sourcedir> <appname> --platform=<platform> --arch=<arch> [optional flags...]

<sourcedir> = diretorio onde esta os codigos
<appname> = nome do aplicativo
<platform> = Para Windows win32, Para MAC darwin, Para Linux linux
<arch> = arquitetura  32bits = ia32 , 64bit = x64
